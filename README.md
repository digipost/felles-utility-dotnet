# felles-utility-project-rider-dotnet

Dette er en fork av difi-felles-utility for å teste Intellij Rider. 

__Det som fungerer er:__
* Bygge
* Kjøre tester

__Det som ikke fungerer er:__
* Steppe inn i tester
* Tester som henter fra Windows Keystore
